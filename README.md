# atmnb
[![Build Status](https://travis-ci.org/qiugits/atmnb.svg?branch=master)](https://travis-ci.org/qiugits/atmnb)\
manaba出席のコマンドラインツール、node.js版です。\
描画しないため、速く処理をすることができます。

## Install

```terminal
npm install atmnb
```

## Usage

```terminal
$ atmnb 1234567
```
